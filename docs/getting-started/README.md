# Getting Started with Source Code Portal

This section provides everything you need to get Source Code Portal up and running.

- **[Running](running.md)** - Spring Boot execution modes and configuration
- **[Configuration](configuration.md)** - Configure GitHub authentication and repository groups

## Prerequisites

Before starting, ensure you have:

- **Java 21 LTS** - Required runtime
- **Maven 3.6+** - Build tool
- **GitHub Access Token** - For API access (see [Configuration](configuration.md))
- **(Optional) Sass** - For frontend development
